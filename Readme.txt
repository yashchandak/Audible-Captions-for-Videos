##Requirements:

Python3
Tensorflow
espeak
gTTS
pillow
numpy
matplotlib
pickle

##Execution instructions

1) activate tensorflow environment
2) execute demo.py on the server
3) execute audio_out.py and cameraClient.py on the rasPi device

4) (Optional) look into other codes for self-contained documentation and additional details. For examples and future updates please check https://github.com/yashchandak/Audible-Captions-for-Videos
